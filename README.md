## 📖 Sobre o projeto

O **SAUDITACRYPTO** é uma landing page construída para demonstrar habilidades em front-end, com foco em um design monocromático limpo, animações suaves e boas práticas de HTML, CSS e JavaScript puro — sem uso de frameworks.

---

## ✨ Destaques

- 🎨 **Design monocromático** (preto e branco) com foco em contraste e minimalismo
- 📱 **Hero section** com mockups de celular animados exibindo carteira cripto, gráfico de saldo e efeito parallax no mouse
- 🗂️ **Cards numerados** (01, 02, 03) destacando os diferenciais da plataforma
- 📈 **Gráfico de mercado interativo** em SVG, com animação de desenho da linha (`stroke-dashoffset`) ao entrar na viewport
- 🕹️ **Tabs de período** (1D / 1W / 1M / 1Y) no gráfico principal
- 🌀 **Scroll reveal animations** via `IntersectionObserver`
- 📌 **Header com efeito de scroll** e menu mobile responsivo (hamburger animado)
- 💫 **Micro-interações**: ripple effect nos botões, hover states, coins flutuantes animadas
- ⚡ **100% Vanilla** — sem frameworks ou dependências externas

---

## 🛠️ Tecnologias utilizadas

- **HTML5** semântico
- **CSS3** (Flexbox, Grid, custom properties, animações)
- **JavaScript** puro (ES6+), sem bibliotecas

cd SauditaCrypto
